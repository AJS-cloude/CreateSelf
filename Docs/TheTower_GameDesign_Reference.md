# The Tower - Idle Tower Defense | 게임 설계 참고 문서

> Tech Tree Games 제작. 타워 디펜스 + 방치형. 동일/유사 게임 제작용 정리.

---

## 1. 개요

- **장르**: 디펜스(타워 디펜스) + 방치형
- **핵심**: 중앙 타워를 지키며 웨이브별로 오는 적을 막는 구조
- **재화**: **현금**(인게임, 웨이브 중 업그레이드용), **동전**(라운드 종료 후 워크샵/업그레이드용), **보석**(슬롯/카드/실험실 해금 등)

---

## 2. 난이도 등급 (Tier)

| 등급 | 코인 배수 | 비고 |
|------|-----------|------|
| 1~14 | 1.0 ~ 17.6 (단계별 상승) | 등급별 **101웨이브** 도달 시 다음 등급 해금 |
| 15~21 | (추가 등급) | 14등급부터 **히트 조건** (페널티) 적용 |

- 각 등급 101웨이브 달성 → 다음 등급 오픈
- 코인 보너스는 등급이 올라갈수록 증가 (성장 가속)

## 4. 궁극의 무기 (Ultimate Weapons)

**파워스톤**으로 해금/업그레이드. (현질, 토너먼트, 이벤트, 주간 임무 등으로 획득)

| 티어 | 이름 | 효과 요약 |
|-----|------|-----------|
| **S+** | 골든 타워 | 킬 시 현금/코인 대폭 증가, 고웨이브 도달 용이 |
| **S** | 데스웨이브 | 화면 적 대부분 타격. 킬 시 체력 증가(연구로 최대 8배) |
| **S-** | 크로노필드 | 범위 내 슬로우 오라. 연구로 범위/지속/추가 피해 |
| **B** | 독의 늪 | 죽은 적이 독장판 생성, 독 지속 피해 |
| **B-** | 내부 지뢰 | 타워 근처 지뢰, 연구로 보스 포함 기절 |
| **C** | 블랙홀 | 범위 밖에 생성, 적 빨아들임 (보스 무효). 이점/리스크 있음 |
| **D** | 연쇄 번개 | 번개가 적 간 튕기며 다수 타격 |
| **F** | 스마트 미사일 | 사거리 가장 먼 적에게 유도 미사일 |
| (추가) | 스포트라이트 | 특정 범위 내 적 피해 증가 (고랭크 메타) |

- 추천 해금 순서: 골든타워 → 블랙홀 → 데스웨이브 → 크로노필드 → 스포트라이트

## 5. 카드

- **장착 슬롯**: 최대 21개 (보석으로 슬롯 해금)
- **승급**: 별 개수로 1→2(3장), 2→3(5장), … 6→7(32장) 등
- **등급**: 일반 / 레어 / 에픽

### 대표 카드 예시

| 등급 | 이름 | 효과 예시 |
|------|------|-----------|
| 일반 | 데미지/체력/공격속도/범위/현금/동전 | 해당 스탯 배율 증가 |
| 일반 | 저속 아우라 | 타워 범위 내 적 속도 감소 |
| 일반 | 치명타율 / 추가방어 / 요새 | 치명타율+, 방어율+, 절대방어 배율 |
| 레어 | 무료 업그레이드 | 웨이브당 무료 업그레이드 확률 +n% |
| 레어 | 내부 오브 | 추가 회전 구체 (연구로 오브 추가) |
| 레어 | 플라즈마 대포 | 보스 체력 n% 떨어뜨릴 때 강력 한 방 |
| 레어 | 치명적 동전 / 웨이브 건너뛰기 | 코인 드랍, 웨이브 스킵+보상 |
| 에픽 | 죽음의 광선 | 접촉 적(보스 제외) 즉사 광선, n초 지속 |
| 에픽 | 에너지 네트 | 보스 행동 정지 그물 |
| 에픽 | 슈퍼 타워 | 일정 시간 타워 피해 배율 증가 |
| 에픽 | 2차 웨이브 | 라운드당 1회 체력 절반으로 부활 + 무적 실드 |
| 에픽 | 에너지 방패 | 한 번의 공격 무시, n분 후 재충전 |
| 에픽 | 광전사(berserker) | 흡수한 데미지 비율만큼 피해 증가 (상한 있음) |

- 코인 파밍용: Coins, Enemy Balance, Critical Coin, Wave Skip, Death Ray, Inner Orbs 등
- 토너먼트용: Enemy Balance, Attack Speed, Extra Defense, Health, Inner Orbs, Death Ray 등

## 6. 실험실 (연구)

- **슬롯**: 최대 5개 (보석으로 Lab 2~5 해금)
- **게임 속도**: 연구로 최대 5배속 등
- **범위/충격파 크기**: 게임 밖에서 연구 단계별로 조절 가능 (예: 0~15단계)
- 연구로 워크샵 수치 추가 상승, 궁극의 무기 강화, 오브 보스 타격 등 다양한 영구 강화

## 7. 특성 (Perks)

- **200웨이브**(연구로 감소 가능)마다 랜덤 특성 선택
- **일반 특성**: 최대체력×1.20, 데미지×1.15, 체력재생×1.60, 동전 보너스×1.15, 바운스샷+2, 이자×1.50, 지뢰 피해×3.50, 오브+1, 무료업그레이드 확률+5%, 방어율+4%, 특성 웨이브 요건 -20%, 궁극무기 랜덤 해금, 게임속도+1, 궁극무기 특전 등 (중복 선택 가능한 것 있음)
- **궁극무기 특전**: 스마트미사일+4, 늪 반경×1.5, 데스웨이브+1웨이브, 골든타워 보너스×1.5 등 (각 1회)
- **트레이드오프**: 타워 피해×1.50 & 보스 체력×8, 코인×1.80 & 타워 체력 -70%, 적 체력 -50% & 회복/흡혈 -90% 등 (선택 시 큰 장단점)

## 8. 토너먼트

- **일정**: 수요/토요 09:00(KST) 시작, 24시간
- **참가**: 토너먼트 티켓 1장 (없으면 1회 광고 or 보석 10개)
- **티어**: 브론즈 → 실버 → 골드 → 플래티넘 → 챔피언. 30명 단위 매칭, 순위에 따라 승격/강등
- **페널티**(실버 이상): 죽음의 광선/오브/플라즈마 피해 감소, n회 피격 면역, 넉백 감소, 적 속도/공격속도 증가, 죽음의 거부 감소, 추가 적, 가시 피해 감소, 에너지보호막 충전 증가, 보스 등장 주기 변경 등

## 9. 핵심 게임 루프

1. **라운드 시작** → 웨이브 진행 (적 스폰, 타워 자동 공격)
2. **웨이브 중**: **현금**으로 공격/방어/유틸리티 업그레이드
3. **웨이브 완료**: 현금/동전 지급, 무료 업그레이드 확률, 회복 패키지/특성 등
4. **보스**: 주기적 등장 (예: 10웨이브마다). 반사 데미지·플라즈마·오브 등으로 처리
5. **타워 파괴** → 라운드 종료, **동전** 정산
6. **라운드 밖**: 동전으로 워크샵, 보석으로 카드/실험실/슬롯 해금

## 10. 초반 설계 팁 (등급 1 목표)

- **경제**: 현금/웨이브를 올려서 “업그레이드 비용 < 이번 라운드에서 벌 수 있는 현금”이 되기 전까지 투자.
  - 예: 남은 웨이브 수 × $4 × (현금 보너스) 이상이 되면 업그레이드 비용이 회수 불가 → 그 전에 끊기.
- **방어**: 체력 + 방어율% + (초반) 절대방어. 반사 데미지로 보스 처리 (보스 3방 버티면 확정 처치).
- **체력 목표**: 받는 데미지의 약 3배 수준 유지 (흡혈 획득 후에는 보스 공격력의 1/2보다 크면 됨).
- **티어**: 101웨이브만 넘기면 다음 등급 오픈이므로, 빠르게 다음 등급을 여는 것이 코인 수급에 유리.

## 11. 심화 전략: 블렌더

- **핵심**: 오브에 닿은 일반 적 즉사 + 반사 데미지로 보스 처리.
- **필수**: 오브 계열 워크샵 풀업, 추가 오브 2개 이상, **사거리 워크샵은 0으로 두고 인게임에서만 60~80m 정도로 조절**.
- **추가**: 공격속도·오브 속도 최대, Orbs Boss Hit 연구, Death Ray 카드, 궁극무기 블랙홀/크로노필드.
- **배치**: 블렌더(오브 일직선), 프로거(오브를 사거리 기준 안/밖으로 분리), 싱귤러리티(오브가 블랙홀 중심 통과, 코인 파밍).

## 12. 구현 시 체크리스트 (요약)

- [ ] 웨이브 기반 스폰 + 보스 주기
- [ ] 재화: 현금(인게임), 동전(라운드 후), 보석
- [ ] 워크샵: 공격/방어/유틸리티 3계열, 단계별 비용·상한
- [ ] 인게임 업그레이드 (현금 소비)
- [ ] 치명타/슈퍼치명타, 멀티샷, 바운스, 넉백, 오브, 충격파, 지뢰, 반사, 흡혈, 벽
- [ ] 회복 패키지, 무료 업그레이드 확률
- [ ] 등급(Tier) + 101웨이브 해금
- [ ] 카드 슬롯·승급·등급
- [ ] 궁극의 무기 + 파워스톤
- [ ] 실험실(연구) + 게임 속도
